## 📌 Descrição do Projeto

O **FeedBook** é um aplicativo mobile que simula uma plataforma de feed social. Seu principal objetivo é praticar e aplicar conceitos fundamentais do desenvolvimento Android nativo com Kotlin, incluindo:

* Estrutura de projetos Android
* Componentes de UI (Jetpack e Material Design)
* RecyclerView para listagem de conteúdos
* Integração de layouts dinâmicos

---

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** Kotlin
* **IDE:** Android Studio
* **Arquitetura:** MVC / Modularização por camadas
* **Principais Recursos Android:**

  * RecyclerView
  * Intents
  * Activities e Fragments
  * View Binding
  * Material Components

---

## 🚀 Como Rodar o Projeto

1. **Clone o repositório:**

```bash
git clone https://github.com/jampus90/FeedBook.git
```

2. **Abra o projeto no Android Studio**

* File → Open → Selecione a pasta do projeto (`FeedBook`).

3. **Configure um emulador ou dispositivo físico**

* Android SDK recomendado: API 30 ou superior.

4. **Execute o projeto (Run > Run 'app')**

---

## 📷 Funcionalidades Principais

* Exibição de um feed de postagens simuladas
* Layout responsivo
* Navegação entre telas (Activities / Fragments)
* Interface amigável utilizando componentes Material Design
